# Project Euromoon - Treinbeheer Applicatie

Dit project is een Java-toepassing ontwikkeld voor het beheren van internationale treinreizen tussen Brussel, Amsterdam en Parijs. De focus ligt op een robuuste invoervalidatie en een duidelijke structuur.

## Gebruikte Technieken
* **Object-Oriented Programming (OOP):** Gebruik van abstracte klassen, overerving en polymorfisme.
* **Data Validatie:** Controle op Rijksregisternummers (prefix-match), datums en String-input via Regex.
* **File I/O:** Automatische generatie van boardinglijsten in .txt formaat.
* **Error Handling:** Gebruik van try-catch blokken om crashes bij foute invoer te voorkomen.

## Installatie en Gebruik
1. Clone de repository naar je lokale machine.
2. Open het project in een IDE (bij voorkeur IntelliJ).
3. Run de `Main.java` klasse in de `euromoon` package.
4. Volg de instructies in de terminal om passagiers te registreren en tickets te verkopen.
